# Data Science Assessment - SQL Challenge

## Overview

## Database Schema

**investor_transactions**
- `investor_id` (INTEGER)
- `sector_id` (INTEGER)
- `no_of_shares` (INTEGER)

**sectors**
- `sector_id` (INTEGER, PRIMARY KEY)
- `sector_name` (TEXT)
